# tim
tim

一个分布式即时通讯（IM）系统后端实现。

技术栈

后端框架: Gin

消息队列: Kafka

数据库: MySQL、MongoDB、Redis

功能概览

完整的后端功能实现

消息记录可加密（可在 ws 包中实现加解密，也可以让客户端处理加密）

文件上传接口可扩展，通过重写 pkg 包下接口可保存到 OSS

验证码功能可扩展

配置

相关配置文件位于：

config.yaml

前端

前端页面暂未实现，后续将补充。
默认模块 默认模块.html 列出了已实现的接口及使用示例。

快速启动

只需要保留 api 和 db ,client文件夹

下载依赖：

go mod tidy


启动服务：

go run .


服务即可运行
